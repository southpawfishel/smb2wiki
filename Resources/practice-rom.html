This is a ROM hack of the USA version of the NES game useful for practicing specific levels with any character. It was created based on https://github.com/kmck/smb2, a disassembly project for SMB2USA.

How to Obtain the Patch

Join the Super Mario Speedrunning Discord and type the ?smb2usapractice command to get links to the most up to date version, or use one of these links:

  • V0.0
  • V0.1
  • V0.2 (stable)
  • V0.3

How to Install

You will need a legally obtained ROM of the game, and you can use an IPS patching tool like this one to apply the patch to your ROM. You are on your own in obtaining the base ROM to use.

Supported Platforms

The practice ROM is known to work on NES and Famicom hardware when using an Everdrive, and on an NES emulator on PC/Mac.

How to Use

Once you have launched the ROM using your platform of choice, start a new game, choose your character, and start the first level. Once you are in 1-1, you can press Select to bring up the practice menu, which will allow you to adjust your health, amount of lives and continues, jump to any level, and choose which screen of that level to warp to when you start it.
