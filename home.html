This wiki is dedicated to information about Super Mario Bros. 2 on the NES and SNES via Super Mario All Stars. The goal of this wiki is to provide a central repository of information on how to run the game for both new runners and experts. If you want to contribute, please send davesies a message in the Super Mario Speedrunning Discord to get editor access.

If you wish to find more general information about Super Mario Bros. 2, please check the Super Mario Wiki. Happy running!
